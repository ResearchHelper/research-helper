# 研究小助手

研究小助手是一款开源的文献管理工具。它有着内置的 PDF 阅读器以及所见即所得的 Markdown 编辑器。你可以很好地用它来管理自己的电子书，文献以及笔记。

# 为研究小助手作贡献

研究小助手用[QUASAR](https://quasar.dev)作为框架。这款软件采用了[Vue.js3](https://vuejs.org)加[Electron](https://www.electronjs.org)的组合。

## 安装依赖

```bash
yarn # 推荐用yarn
# 或者
npm install
```

## 开始开发吧

```bash
yarn dev # 开发模式
yarn build # 编译测试版
```
